# QTUN Tunneling Changelog

#### v1.0.0 - 01-04-2026
- **Initial Release**: Proyek QTUN Tunneling resmi diluncurkan.
- **Dynamic Workers**: Implementasi pemilihan jumlah worker otomatis berdasarkan Core CPU.
- **Magisk Integration**: Penambahan tombol **Action** untuk kontrol ON/OFF manual.
- **Auto-Start**: Script `service.sh` untuk menjalankan tunnel otomatis setelah booting.
- **Safety**: Penambahan `uninstall.sh` untuk pembersihan total folder `/data/adb/QTUN`.
- **Smart Config**: Fitur backup/restore `config.json` otomatis saat update modul.
